# Video-Audio-downloader-editor-Kecheenok

# Hi, my name is Kxnvra. I make/vibecode stuff. This for a brief explanation

Kecheenok, is audio/video downloader and trimmer, it can download videos or audios from 90% of websites. It run entirely in python and it uses yt-dlp and ffmpef engines for it to work. Essentialy, my entire program is just ui for these other engines. 
Fun fact, the name of the file comes from a nickname of my friend, that made a lot of things for me, so for his honor, i named it after him.
Use it wisely, cuz yt-dlp itself violates youtubes EULA.
